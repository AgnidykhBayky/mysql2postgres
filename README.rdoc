= mysql-to-postgres - MySQL to PostgreSQL Data Translation

JRuby gem/project for converting mysql database to postgresql.

== Origination & Direction

The project is a fork of https://github.com/maxlapshin/mysql2postgres - support for C based extensions
have been removed and JRuby specific code has been rudimentally implemented.  The gem has also been
made more Rails centric through the use ActiveRecord's underpinnings.
